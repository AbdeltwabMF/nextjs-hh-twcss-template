# A Next.js Template for Dapp Developer


## Start development server

```bash
npm run dev
```


## Build the app for production

```bash
npm run build
```


## Run the tests

```bash
npm run test
```


## Deploy the app to vercel

for Preview deployment:

```bash
vercel
```

for Production deployment:

```bash
```bash
vercel --prod
```
